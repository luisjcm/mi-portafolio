# 💻 Personal Portfolio & Tech Showcase

Mi portafolio profesional y plataforma de experimentación técnica, diseñado bajo un enfoque modular y optimizado para el rendimiento.

## 🚀 Tecnologías y Herramientas
- **Core Frontend:** HTML5 semántico, Vanilla JavaScript (ES6+).
- **Estilos y Maquetación:** Tailwind CSS (diseño responsivo y utilitario).
- **Entorno de Construcción:** Vite para un bundling ultra rápido y eficiente.
- **Integraciones:** Arquitectura híbrida que conecta un micro-frontend de React para agentes de IA autónomos (Cubot).

## ⚡ Características Destacadas
- **Rendimiento Óptimo:** Estructura ligera enfocada en buenas prácticas de SEO y accesibilidad.
- **Modularidad:** Separación limpia entre los estilos globales y las lógicas inyectadas en el DOM.
- **Diseño Mobile-First:** Adaptabilidad completa a cualquier resolución de pantalla.
